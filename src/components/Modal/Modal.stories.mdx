# Como o modal funciona?

• Os modais são construídos com HTML, CSS e JavaScript. Eles são posicionados sobre todo o resto do documento e removem a rolagem < body >para que o conteúdo modal role.
• Clicar no “pano de fundo” do modal fechará automaticamente o modal.
• As funções do modais monitoram todos os
