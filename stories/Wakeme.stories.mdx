# Aqui estão os arquivos bases para você começar a usar o Design System CRM Educacional

No Design System armazenamos nossas documentações, componentes e orientações para uma melhor utilização de componentes na criação de nossos produtos digitais. Este documento auxilia designers e desenvolvedores a criarem produtos e experiências digitais consistentes e padronizadas, garantindo práticas de design unificadas e implementação do código eficiente.
